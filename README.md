## Client Repository 

# To install Client app on your local machine, kindly follow these steps.

- from command line, git clone https://github.com/hamdymagdy26/client using command line : <br> 
( git clone git@github.com:hamdymagdy26/client.git )

- create .env file with your database configurations.

- run composer install.

- create a database in your phpmyadmin, for examlpe : client

- run php artisan migrate
